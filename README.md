## 📘 项目简介

`fanqienovel-decryptor` 是一个用于**解密并下载番茄小说内容**的 Python 工具。  
通过逆向番茄小说的字体加密机制，提取真实章节文本，并支持通过小说 ID 批量下载为 `.txt` 格式。

---

## 🔧 功能特点

- ✅ 自动解析并解密字体映射（反反爬机制）
- ⬜ 支持通过小说 ID 批量下载
- ⬜ 输出为纯文本 `.txt` 文件
- ⬜ 可作为脚本或模块集成使用
